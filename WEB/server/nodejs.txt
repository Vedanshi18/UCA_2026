Go into server folder
// npm init (initialises a server)... asks a couple of ques
                                      (version : 1.0.0 => major.minor.patch version)
                                      index.html is entry point for html, index.js for js

Create index.js manually in server folder
Run using -> node index.js  (Ran an index.js file with node being the runtime environment for js)

Utility provided need to be imported using -> require or import
    For using import, we need to set type = module in package.json file
    Module -> functions imported into an object & can be exported as a module to make it reusable, ex: http is a module/object that exports a reusable method createServer()
